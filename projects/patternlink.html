How it works

  1. Draw any pattern on the grid, like a phone lock screen.
  2. Each node is numbered left-to-right, top-to-bottom. Your path becomes a code, e.g. 1-5-9-8-7.
  3. Tap Generate link to get a URL with the pattern baked in.
  4. Send the code or the link. Opening the link replays the pattern automatically — full trace, or one step at a time.
