08Mobile Toolbar — Above Keyboard

The toolbar that stays at the bottom and moves up when the keyboard appears. Two ingredients: position:fixed + the right viewport meta.

/* ① viewport meta — REQUIRED in <head> */
<meta name="viewport"
  content="width=device-width, initial-scale=1,
           interactive-widget=resizes-content">

/* ② CSS */
.toolbar {
  position: fixed;
  bottom: 0;  left: 0;  right: 0;
  display: flex;
  gap: 8px;
  align-items: center;
  padding: 10px 16px;
  /* ↓ add space for iPhone home indicator */
  padding-bottom: calc(10px + env(safe-area-inset-bottom));
  background: var(--surface);
  border-top: 1px solid var(--border);
  z-index: 100;
}

.toolbar input {
  flex: 1;            /* fills space beside button */
  font-size: 16px;   /* prevents iOS zoom-on-focus! */
  padding: 10px;
}

/* ③ so content isn't hidden behind toolbar */
body { padding-bottom: 80px; }
💡 interactive-widget=resizes-content shrinks the viewport when the keyboard opens — so your fixed bar naturally moves up. Without it the bar sits behind the keyboard.
⚠ font-size: 16px on inputs is critical for iOS — anything smaller triggers an automatic zoom that breaks your layout.

09Themes & Accents — CSS Variables + JS

Define colors once on :root. Use them everywhere with var(--name). Change your whole app's look with one JS line.

/* CSS — define on :root (= <html>) */
:root {
  --bg:      #0d0d10;
  --surface: #16161c;
  --text:    #e2e0dc;
  --accent:  #f97316;   /* change this = recolor everything */
}
/* Light theme class */
.light { --bg: #f0ede8; --text: #1a1a20; }

/* Use them */
body   { background: var(--bg); color: var(--text); }
button { background: var(--accent); }

/* JS — toggle dark/light */
document.body.classList.toggle('light');

/* JS — change accent color */
document.documentElement.style
  .setProperty('--accent', '#7c3aed');
